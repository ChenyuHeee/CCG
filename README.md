# CCG - C Code Golf 挑战平台

本仓库包含 C Code Golf 挑战平台的移动端开发项目。

## 📱 iOS 应用

查看 [CCGApp](./CCGApp/) 文件夹获取 iOS 原生应用的完整实现。

### 快速预览

CCG iOS App 是为 [C Code Golf 挑战平台](https://chenyuheee.github.io/c/competition) 开发的原生 iOS 客户端，提供：

- 🎨 美观的 SwiftUI 界面设计
- 💻 内置 C 代码编辑器
- 🏆 实时排行榜展示
- 📊 积分统计与天梯系统
- ⚡ 流畅的原生体验

### 技术栈

- Swift 5.9 + SwiftUI
- iOS 17.0+
- MVVM 架构
- Async/Await 异步编程

查看 [CCGApp/README.md](./CCGApp/README.md) 了解详细信息。
